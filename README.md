
# ASP.NET Core Web API CRUD

## 📖 Project Overview
This project is a CRUD Web API developed using ASP.NET Core Web API, Entity Framework Core, and SQL Server. It provides RESTful APIs to Create, Read, Update, and Delete student records.

---

## 🚀 Technologies Used

- ASP.NET Core Web API
- C#
- Entity Framework Core
- SQL Server
- Visual Studio 2022

---

## ✨ Features

- Get All Students
- Get Student By Id
- Create Student
- Update Student
- Delete Student

---

## 📌 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /api/StudentAPI | Get All Students |
| GET | /api/StudentAPI/{id} | Get Student By Id |
| POST | /api/StudentAPI | Create Student |
| PUT | /api/StudentAPI/{id} | Update Student |
| DELETE | /api/StudentAPI/{id} | Delete Student |

---

## ⚙️ How to Run

1. Clone the repository.
2. Open the project in Visual Studio.
3. Update SQL Server connection string.
4. Run the project.
5. Test APIs using Swagger.
